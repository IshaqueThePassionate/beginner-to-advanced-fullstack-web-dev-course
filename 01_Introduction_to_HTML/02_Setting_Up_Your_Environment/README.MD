
# Module 01.2: Setting Up Your Environment

Now that we know how the web works, it's time to prepare our tools. To build websites, you only need two things: something to **write** code in and something to **see** the results in.

---

## 1. Prerequisites

The best thing about HTML is that there are **zero prerequisites**.

* You don't need to know any other programming language.
* You don't need a super-powerful computer.
* It is often the very first step anyone takes into the world of coding.

---

## 2. The Tools You Need

### A. The HTML Editor

An editor is where you write your code.

* **The Reality:** You *can* write HTML in a simple app like Notepad (Windows) or TextEdit (Mac).
* **The Better Way:** Professional developers use "Code Editors" or "IDEs" because they provide features like auto-completion and error highlighting.
* **Popular Choices:** * **Visual Studio Code (VS Code):** Lightweight, open-source, and the industry standard. (This is what we will use).
* **Sublime Text:** Very fast and clean.
* **Online Editors:** CodePen or JSFiddle (great for quick testing without installing anything).



### B. The Web Browser

The browser reads your code and displays the website.

* **Our Choice:** **Google Chrome** is highly recommended for its excellent developer tools.
* **Other Options:** Firefox, Microsoft Edge, and Safari.

---

## 3. Installing Visual Studio Code (Step-by-Step)

VS Code is widely considered the best free editor on the market. Follow these steps to set it up:

1. **Search:** Go to Google and search for **"Visual Studio Code download."**
2. **Download:** Select the version for your Operating System (Windows, macOS, or Linux).
3. **Install:** Run the installer. On Windows, make sure to check the boxes that say **"Add to PATH"** and **"Add 'Open with Code' action"**—these are very helpful shortcuts.

## 4. The Superpower: Live Server Extension

By default, every time you change your code, you have to manually refresh your browser to see the changes. We can automate this using an **Extension**.

### What is Live Server?

It is a tool that launches a local development server on your computer with a **"Live Reload"** feature. The moment you save your code, the browser updates automatically.

### How to Install it:

1. Open VS Code.
2. Click on the **Extensions icon** on the left sidebar (the icon looks like four squares).
3. Search for **"Live Server"** (by Ritwick Dey).
4. Click **Install**.
